### Project Description
SlangSwap ‚Äì A fun web app that translates slang words from one Kerala district into the slang of all other districts, complete with pronunciation and examples.
Currently, the database contains only a small set of slang words for demo purposes ‚Äî but it can be expanded in the future to include hundreds more.

### The Problem (that doesn't exist)
Keralites love their slang, but sometimes slang from one district sounds like alien language to another district.
